CAMP Wuhan Version 42

CAMP is a measurement program for MATLAB and Arduino. It is designed to with for the wind tunnels set up at Jianghan University (江汉大学) and at the Technical University of Kaiserslautern. The documentation of CAMP contains further information.

You can find the most recent version on github under sdschmidt/CAMP. A packaged *.zip archive containg updated information but no configuration file is available here. Please remember setting up a configuration file and don't edit classes in include without deactivating the update in start.m:13: 

autoupdate = 0; 
% autoupdate = 1;

Current Features

  • create and load measurements
  • perform line, field measurements or specify any other shapes of points
  • view old measurements and take notes
  • interrupt and resume measurements
    • continue mesurements where you left off
    • skip already measured points
  • tracking of the probe position allows for absolute and relative movements
  • different plotting options
  • easy adaption of the postprocessing method
  • backend and frontend separated
  • self-updating
  • configuration file
  • rudimentary working velocity pid controller

Upcomming Features

  • data management overhaul for more efficient backend and GUI updates
  • velecity control with combined feedforward/feedback methods
